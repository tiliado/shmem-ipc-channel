Channel

Object Hierarchy:

Description:

public class Channel

A binary message channel using POSIX shared memory and semaphores. It supports server ( Mode.SERVER)/client (Mode.CLIENT ) mode, requests (request, set_request_callback) with corresponding responses and notifications without a response (set_notification_callback).
