set_request_callback

Description:

public void set_request_callback (owned RequestCallback callback)

Set callback to be called to handle incoming requests.

The data of the request must be used immediately or a copy must be made. The callback is executed in the thread the send_receive method is called in.

Parameters:

callback

The request callback.
