Shmem

Description:

public Shmem (string name, ulong size, bool create, bool discard) throws Error

Create or open POSIX shared memory.

Use pointer and get_buffer to access the shared memory region with the actual size size. Then close the shared memory when no longer needed. It is also automatically closed in destructor, but you should not count on it. Explicit is better than implicit in case of resources.

Parameters:

name

The shared memory name. It must contains only a single `/` at the very beginning and not exceed 255 characters.

size

The shared memory size. It may be 0 if `create` is `true` and the actual size will used instead and available as size.

create

Whether to create new shared memory region or to open an existing one.

discard

Whether to discard the shared memory region or let it be alive upon close.

Exceptions:

Error

on failure: Error.INVALID_NAME, Error.INVALID_SIZE, Error.SHM_OPEN_FAILED.
