Shared memory channel errors


Namespace: Shmch
Package: shmchannel

Content:

Error codes:

  • ALREADY_OPEN - The resource has already been opened.
  • CLOSED - The resource is closed.
  • INVALID_NAME - The name of shared memory is not valid.
  • INVALID_SIZE - The size of shared memory is not valid.
  • SHM_OPEN_FAILED - Failed to open shared memory or set its size up.
  • SHM_CLOSE_FAILED - Failed to close shared memory.
  • RESOURCE_LIMIT - When the requested action cannot be performed because of a resource limit.
