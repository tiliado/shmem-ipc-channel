set_notification_callback

Description:

public void set_notification_callback (owned DataCallback callback)

Set callback to be called to handle incoming notification.

The data of the notification must be used immediately or a copy must be made. The callback is executed in the thread the send_receive method is called in.

Parameters:

callback

The notification callback.
