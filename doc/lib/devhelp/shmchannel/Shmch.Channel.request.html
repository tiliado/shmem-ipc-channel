request

Description:

public void request (uint8[] data, owned DataCallback response_callback) throws Error

Send a request.

The callback is executed in the thread the send_receive method is called in.

Parameters:

data

The request data.

response_callback

The callback to be called when a response arrives.

Exceptions:

Error

on failure: Error.RESOURCE_LIMIT, Error.SHM_OPEN_FAILED, Error.SHM_CLOSE_FAILED.
