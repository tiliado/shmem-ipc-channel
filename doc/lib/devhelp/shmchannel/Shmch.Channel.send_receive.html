send_receive

Description:

public bool send_receive (bool wait) throws Error

Send and receive messages.

This method does the heavy lifting and should be called periodically, e. g. as an idle callback in an event loop. Otherwise, no messages are sent nor received.

Parameters:

wait

Whether to wait if the channel is currently locked by the other side. It may block then.

Returns:

`True` if any data have been received or sent.

Exceptions:

Error

on failure: Error.SHM_OPEN_FAILED, Error.SHM_CLOSE_FAILED, Error.CLOSED.
